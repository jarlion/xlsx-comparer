## Xlsx 对比工具

对比两个 .xlsx 文件的异同，并且用颜色显示

## 安装
```bash
npm i
```

## 编译 TS 文件
```bash
# 编译所有 *.ts 文件
tsc
# 或者持续编译
tsc -w
```

## 使用命令行对比
```bash
# xc 对比命令
# -s (-source) 需要对比的源文件(*.xlsx)路径
# -t (-target) 用于对比的目标文件(*.xlsx)路径
# -k (-key) 主键名
node ./bin/server/index.js xc -s test/a.xlsx -t test/b.xlsx -k A
```
